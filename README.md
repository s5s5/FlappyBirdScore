# Flappy Bird 成绩生成器

练手之作，本是玩玩，但众欢喜，固优化之，请赐教！

## 功能

- 输入任意数字生成 Flappy Bird 分数截图
- 根据分数自动显示不同等级的奖牌（铜、银、金、白金）
- 支持下载生成的图片
- 纯前端实现，使用 Canvas 生成图片

## 项目结构

```
FlappyBirdScore/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式
├── js/
│   └── app.js          # 前端逻辑
└── assets/
    ├── fonts/          # 字体文件
    └── images/         # 图片资源
```

## 本地预览

直接用浏览器打开 `index.html` 即可
